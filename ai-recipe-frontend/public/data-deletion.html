사용자 데이터 삭제 안내

recipfree는 로그인용 OAuth 프로필(이메일/닉네임/프로필 이미지 등 최소 정보)만 보관하며, 사용자의 요청 시 삭제합니다.

삭제 요청 방법

  1. 제목에 “데이터 삭제 요청”을 적어 support@recipfree.com 으로 이메일을 보내주세요.
  2. 이메일 본문에 아래를 포함해 주세요:
    • 로그인 제공자(예: Facebook/Google/Naver/Kakao)
    • 해당 제공자 ID(가능하면)
    • 서비스에서 사용한 이메일 주소

요청을 확인한 뒤 관련 법령이 허용하는 범위에서 지체 없이 삭제 처리합니다.
